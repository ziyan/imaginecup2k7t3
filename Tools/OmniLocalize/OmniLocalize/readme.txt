Note: ResGen.exe must be in the same directory as OmniLocalize.exe for this program to work.

ResGen.exe is a .NET SDK tool and can be found somewhere like
C:\Program Files\Microsoft Visual Studio 8\SDK\v2.0\Bin



Basic usage:
1. English Site Resources -> Excel
Leave the Culture Code blank, choose "Resource Files to CSV (Excel)"

2. Other Language (e.g. Chinese) Site Resources -> Excel
Enter the culture code (e.g. "zh-CN", choose "Resource Files to CSV (Excel)"

3. How to localize Omni into a new language
Same steps as 1 (begin from English).

4. Taking a localized file, and integrating it with the site
Not implemented yet.
